# The Agentic AI PM Stack 🧠⚙️

An interactive, living architecture map designed to cut through the noise of the modern AI ecosystem. 

Navigating the fragmented landscape of autonomous agents, orchestration frameworks, and governance tools shouldn't require a PhD in marketing jargon. This dashboard organizes the chaos, illustrating exactly how these platforms plug into the modern product development lifecycle so teams can confidently decide what to build next.

## 🏗️ Architecture & Tech Stack

This project is built with a focus on speed, modularity, and clean UI/UX:
*   **Framework:** React (Vite)
*   **Styling:** Tailwind CSS with custom glassmorphic and dark-mode neon design tokens
*   **UI Components:** shadcn/ui & Radix Primitives
*   **Routing & State:** React Router
*   **Deployment:** Vercel (CI/CD integrated)

## 💡 The Product Philosophy

There is a massive gap between "AI as a buzzword" and "AI as a scalable product feature." 

This stack was built to solve a specific problem: product teams need to evaluate non-deterministic systems, map out multi-agent workflows, and understand context windows without getting lost in the weeds. By mapping out orchestration frameworks (like LangChain or LlamaIndex) alongside observability tools and enterprise governance platforms, this dashboard serves as a functional compass for AI-Native product development.
